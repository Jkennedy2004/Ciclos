Bienvenido a la demostración de DevSecOps

Esta es una aplicación web de prueba con vulnerabilidades intencionadas.
El objetivo es mostrar cómo las herramientas de seguridad integradas en un pipeline de CI/CD pueden detectar estos problemas.

Puedes interactuar con los siguientes endpoints para ver las vulnerabilidades en acción:

  • **Vulnerabilidad de inyección de comandos:** /insecure_route?command=ls
  • **Simulación de inyección SQL y cookie insegura:** Usa el siguiente formulario de inicio de sesión
